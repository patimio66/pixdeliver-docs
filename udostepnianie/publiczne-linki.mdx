## Przegląd

Każda aktywna galeria w PixDeliver ma swój unikalny link, który możesz udostępnić klientom. Link prowadzi do profesjonalnej strony galerii, gdzie klienci mogą przeglądać i pobierać zdjęcia. System oferuje dwa formaty linków - podstawowy i z subdomeną.

**Kluczowe informacje:**

- Link działa tylko dla aktywnych galerii (opłaconych)
- Możesz użyć linku podstawowego lub z subdomeną
- Link można udostępniać dowolną metodą (email, SMS, messenger)
- Nieograniczona liczba osób może otworzyć galerię
- Link jest ważny przez cały rok (lub do wygaśnięcia galerii)

---

## Formaty linków

### 1. Link podstawowy (root domain)

```
https://pixdeliver.com/g/[slug-galerii]
```

**Przykłady:**

- `https://pixdeliver.com/g/wesele-anna-tomasz`
- `https://pixdeliver.com/g/sesja-rodzinna-kowalskich`
- `https://pixdeliver.com/g/event-firmowy-2024`

**Zalety:**

- Działa zawsze, nawet bez subdomeny
- Krótki i prosty
- Uniwersalny dla wszystkich użytkowników

**Wady:**

- Nie zawiera Twojego brandu/nazwy
- Mniej profesjonalny
- Każdy fotograf używa tej samej domeny głównej

---

### 2. Link z subdomeną

```
https://[twoja-subdomena].pixdeliver.com/[slug-galerii]
```

**Przykłady:**

- `https://jan-kowalski.pixdeliver.com/wesele-anna-tomasz`
- `https://fotografia-marek.pixdeliver.com/sesja-rodzinna`
- `https://studio-light.pixdeliver.com/event-firmowy`

**Zalety:**

- ✨ Profesjonalny wygląd
- 🎨 Zawiera Twoją nazwę/brand
- 📈 Buduje rozpoznawalność
- 🌟 Łatwiejszy do zapamiętania dla klientów
- 💼 Wygląda bardziej personalnie

**Wady:**

- Wymaga wcześniejszego ustawienia subdomeny w profilu
- Trochę dłuższy link

<Tip>
  **Rekomendacja:** Używaj linku z subdomeną dla profesjonalnego wizerunku. To bezpłatna funkcja dostępna na każdym planie — pełen przewodnik: [Własna subdomena](/udostepnianie/subdomeny).
</Tip>

## Jak skopiować link do galerii

### Metoda 1: Z panelu galerii

**Instrukcja:**

1. Zaloguj się do PixDeliver
2. Otwórz wybraną galerię
3. Znajdź przycisk **Kopiuj link publiczny**
4. Kliknij przycisk
5. Link zostanie skopiowany do schowka

**Rezultat:** Link jest gotowy do wklejenia w email, SMS lub komunikator.

### Metoda 2: Ręczne skopiowanie z paska adresu

**Instrukcja:**

1. Otwórz galerię jako klient (Podgląd galerii)
2. Zaznacz adres URL w pasku przeglądarki
3. Kliknij prawym przyciskiem → **Kopiuj**
4. Lub użyj skrótu Ctrl+C (Windows) / Cmd+C (Mac)

## Jak wysłać link klientom

### Email

Skorzystaj z gotowego szablonu lub przygotuj własny!

**Przykładowa wiadomość:**

```
Temat: Twoje zdjęcia są gotowe! 📸

Cześć [Imię]!

Dziękuję za wspaniałą sesję! Twoje zdjęcia są już gotowe do obejrzenia i pobrania.

🔗 Link do galerii:
<Tu wklej link do swojej galerii>

[Jeśli galeria ma hasło:]
🔒 Hasło: sesja2024

💾 Możesz:
- Przeglądać zdjęcia online
- Pobierać pojedyncze zdjęcia
- Pobrać całą galerię jako ZIP
- Udostępniać link rodzinie i znajomym

Zdjęcia będą dostępne przez rok.

Jeśli masz jakiekolwiek pytania, śmiało pisz!

Pozdrawiam,
[Twoje imię]
```

<Tip>
  **Profesjonalny dodatek:** Dołącz krótką instrukcję, jak przeglądać i pobierać zdjęcia. Twoi klienci to docenią!
</Tip>

### SMS

Idealny dla szybkiego udostępnienia.

**Przykładowa wiadomość:**

```
Cześć! Twoje zdjęcia są gotowe:
<Tu wklej link do swojej galerii>

Hasło: ania2024

Pozdrawiam!
```

### Messenger / WhatsApp

**Przykładowa wiadomość:**

```
Hej! 👋

Twoje zdjęcia są gotowe! 🎉

Link: https://jan-kowalski.pixdeliver.com/wesele-nowak
Hasło: wesele2024

Zdjęcia są dostępne przez rok 😊
```

<Warning>
  Unikaj publikowania linków publicznie, jeśli galeria zawiera prywatne zdjęcia. Wysyłaj linki przez wiadomości prywatne.
</Warning>

## Udostępnianie linku wielu osobom

Link do galerii może być otwarty przez nieograniczoną liczbę osób.

**Scenariusze:**

- **Wesele** - wyślij link parze młodej, a oni mogą udostępnić go gościom
- **Wydarzenie firmowe** - wyślij link organizatorowi, który przekaże go uczestnikom
- **Sesja rodzinna** - organizatorzy mogą udostępnić link całej rodzinie

<Info>
  **Brak limitów:** Nieograniczona liczba osób może otworzyć i przeglądać galerię. Nie ma limitu wyświetleń ani pobrań.
</Info>

## Co widzą klienci po otwarciu linku

### Galeria bez hasła

1. Klient klika link
2. Natychmiastowo widzi stronę galerii z:
   - Nazwą galerii
   - Okładką (tzw. cover photo)
   - Opisem (jeśli dodany)
   - Siatką miniatur zdjęć
   - Przyciskiem "Pobierz całą galerię"

### Galeria z hasłem

1. Klient klika link
2. Widzi ekran z prośbą o hasło:
   - Pole do wpisania hasła
   - Przycisk "Otwórz galerię"
3. Po wpisaniu poprawnego hasła:
   - Galeria się otwiera
   - Hasło jest zapamiętane w przeglądarce
   - Nie trzeba go wpisywać ponownie (dopóki przeglądarka nie zostanie zamknięta)

## Zmiana linku

### Zmiana slug (adresu URL)

Możesz zmienić slug galerii w ustawieniach, ale:

<Warning>
  **Uwaga!** Po zmianie slug:

  - Stary link przestaje działać natychmiastowo
  - Musisz wysłać klientom nowy link
  - Nie ma przekierowania ze starego linku na nowy
  - **Zmieniaj slug tylko przed udostępnieniem galerii!**
</Warning>

**Instrukcja:**

1. Przejdź do **Ustawienia galerii**
2. Zmień pole **Slug**
3. Zapisz zmiany
4. Skopiuj nowy link
5. Wyślij go klientom ponownie

### Zmiana subdomeny

Zmiana subdomeny wpływa na wszystkie Twoje galerie:

<Warning>
  **Uwaga!** Po zmianie subdomeny:

  - Wszystkie stare linki z subdomeną przestają działać
  - Musisz zaktualizować linki u wszystkich klientów
  - **Nie zmieniaj subdomeny po wysłaniu linków!**
</Warning>

## Śledzenie dostępu

PixDeliver automatycznie śledzi, kto i kiedy odwiedza galerię.

**Co jest śledzone:**

- Liczba wyświetleń galerii
- Unikalni odwiedzający
- Data i godzina dostępu
- Źródło dostępu (link podstawowy vs subdomena)
- Liczba pobrań (całej galerii i pojedynczych zdjęć)

**Zobacz statystyki:**

1. Otwórz galerię w panelu
2. Kliknij **Statystyki** lub **Zobacz statystyki**
3. Zobaczysz pełne dane analityczne

<Info>
  Przejdź do sekcji [Statystyki galerii](/statystyki/pulpit-glowny) aby dowiedzieć się więcej o analityce.
</Info>

## Bezpieczeństwo linków

### Losowość slug

Chociaż możesz wybrać własny slug, dobrze jest dodać element losowości dla większego bezpieczeństwa:

**Przykłady:**

- ✅ `wesele-ania-xk4mp` (slug + losowe znaki)
- ✅ `sesja-kowalskich-2024-a8k` (slug + rok + kod)
- ❌ `wesele-ania` (zbyt przewidywalne)
- ❌ `test` (bardzo ogólne)

<Tip>
  Jeśli galeria zawiera wrażliwe zdjęcia, użyj trudnego do odgadnięcia slug + hasło dla maksymalnego bezpieczeństwa.
</Tip>

### HTTPS

Wszystkie linki PixDeliver używają szyfrowanego połączenia HTTPS:

- ✅ Bezpieczne przesyłanie danych
- ✅ Ochrona przed podsłuchem
- ✅ Certyfikat SSL

## Wygaśnięcie linku

Link przestaje działać w następujących sytuacjach:

### 1. Galeria wygasła

Po upływie roku od opłacenia galerii (lub po wygaśnięciu Konta Studio), i bez jej przedłużenia, klienci otwierający link widzą stronę informującą o wygaśnięciu galerii — z prośbą o kontakt z fotografem.

**Rozwiązanie:** Przedłuż galerię ([Odnowienie galerii](/platnosci/odnowienie-galerii)) albo odnów subskrypcję ([Subskrypcja Konta Studio](/platnosci/subskrypcja-konta)).

### 2. Galeria została usunięta

Jeśli usunąłeś galerię, link przestaje działać natychmiastowo.

### 3. Zmiana slug

Po zmianie slug stary link nie działa (bez przekierowania).

## Wskazówki i najlepsze praktyki

✅ **Zalecane:**

- Używaj linku z subdomeną dla profesjonalnego wizerunku
- Dodawaj instrukcje w wiadomości z linkiem
- Dołączaj hasło (jeśli ustawione)
- Informuj klientów o ważności galerii (1 rok)
- Wysyłaj link dopiero po aktywacji galerii
- Sprawdź link w podglądzie przed wysłaniem
- Zachowaj kopię wysłanych linków (dla siebie)
- Dodaj emoji do wiadomości dla lepszego wrażenia 📸✨

❌ **Unikaj:**

- Zmiany slug po wysłaniu linku klientom
- Publikowania linków publicznie (jeśli galeria prywatna)
- Wysyłania linku do nieopłaconej galerii
- Wysyłania linku bez hasła (jeśli jest wymagane)
- Używania bardzo krótkich, przewidywalnych slug bez hasła
- Ignorowania komunikatów o wygasających galeriach

## Rozwiązywanie problemów

**Problem:** Klient nie może otworzyć linku. **Rozwiązanie:**

1. Sprawdź, czy galeria jest aktywna (opłacona)
2. Sprawdź, czy link został skopiowany poprawnie (bez obcięcia)
3. Upewnij się, że galeria nie wygasła
4. Sprawdź, czy nie zmieniłeś slug po wysłaniu linku
5. Wyślij link ponownie (skopiuj świeży z panelu)

**Problem:** Klient mówi, że link wymaga hasła, ale nie wysłałem hasła. **Rozwiązanie:**

- Sprawdź ustawienia galerii - czy hasło jest ustawione?
- Jeśli tak, wyślij hasło klientowi
- Jeśli nie powinno być hasła, usuń je w ustawieniach galerii

**Problem:** Link działa, ale strona jest pusta (brak zdjęć). **Rozwiązanie:**

1. Sprawdź, czy zdjęcia zostały poprawnie przetworzone
2. Zobacz czy nie ma komunikatu o nieudanych konwersjach
3. Odśwież stronę (Ctrl+F5)
4. Spróbuj w innej przeglądarce
5. Sprawdź, czy zdjęcia są widoczne w panelu zarządzania

**Problem:** Wysłałem link, ale klient go zgubił. **Rozwiązanie:**

- Skopiuj link ponownie z panelu
- Wyślij go jeszcze raz klientowi
- Zachowuj kopie wysłanych linków w swoich emailach
