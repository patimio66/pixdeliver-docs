## Überblick

Jede aktive Galerie in PixDeliver hat ihren eigenen, einzigartigen Link, den Sie mit Ihren Kunden teilen können. Der Link führt zu einer professionellen Galerieseite, auf der Ihre Kunden die Fotos ansehen und herunterladen können. Das System bietet zwei Linkformate – das einfache Format und das Format mit Subdomain.

**Wichtige Informationen:**

- Der Link funktioniert nur für aktive (bezahlte) Galerien
- Sie können den einfachen Link oder den Link mit Subdomain verwenden
- Der Link lässt sich auf beliebigem Weg teilen (E-Mail, SMS, Messenger)
- Eine unbegrenzte Anzahl von Personen kann die Galerie öffnen
- Der Link ist ein ganzes Jahr lang gültig (oder bis zum Ablauf der Galerie)

---

## Linkformate

### 1. Einfacher Link (Root-Domain)

```
https://pixdeliver.com/g/[slug-galerii]
```

**Beispiele:**

- `https://pixdeliver.com/g/wesele-anna-tomasz`
- `https://pixdeliver.com/g/sesja-rodzinna-kowalskich`
- `https://pixdeliver.com/g/event-firmowy-2024`

**Vorteile:**

- Funktioniert immer, auch ohne Subdomain
- Kurz und einfach
- Universell für alle Nutzer

**Nachteile:**

- Enthält nicht Ihre Marke bzw. Ihren Namen
- Wirkt weniger professionell
- Jeder Fotograf verwendet dieselbe Hauptdomain

---

### 2. Link mit Subdomain

```
https://[twoja-subdomena].pixdeliver.com/[slug-galerii]
```

**Beispiele:**

- `https://jan-kowalski.pixdeliver.com/wesele-anna-tomasz`
- `https://fotografia-marek.pixdeliver.com/sesja-rodzinna`
- `https://studio-light.pixdeliver.com/event-firmowy`

**Vorteile:**

- ✨ Professionelles Erscheinungsbild
- 🎨 Enthält Ihren Namen bzw. Ihre Marke
- 📈 Stärkt den Wiedererkennungswert
- 🌟 Leichter für Kunden zu merken
- 💼 Wirkt persönlicher

**Nachteile:**

- Erfordert das vorherige Einrichten einer Subdomain im Profil
- Der Link ist etwas länger

<Tip>
  **Empfehlung:** Verwenden Sie den Link mit Subdomain für einen professionellen Auftritt. Das ist eine kostenlose Funktion, die in jedem Plan verfügbar ist – die vollständige Anleitung finden Sie hier: [Eigene Subdomain](/de/teilen/subdomains).
</Tip>

## So kopieren Sie den Galerie-Link

### Methode 1: Aus dem Galerie-Bereich

**Anleitung:**

1. Melden Sie sich bei PixDeliver an
2. Öffnen Sie die gewünschte Galerie
3. Suchen Sie die Schaltfläche **Öffentlichen Link kopieren**
4. Klicken Sie auf die Schaltfläche
5. Der Link wird in die Zwischenablage kopiert

**Ergebnis:** Der Link ist bereit, um in eine E-Mail, SMS oder einen Messenger eingefügt zu werden.

### Methode 2: Manuelles Kopieren aus der Adressleiste

**Anleitung:**

1. Öffnen Sie die Galerie als Kunde (Galerie-Vorschau)
2. Markieren Sie die URL in der Adressleiste des Browsers
3. Klicken Sie mit der rechten Maustaste → **Kopieren**
4. Oder verwenden Sie das Tastenkürzel Strg+C (Windows) / Cmd+C (Mac)

## So senden Sie den Link an Ihre Kunden

### E-Mail

Nutzen Sie eine fertige Vorlage oder erstellen Sie Ihre eigene!

**Beispielnachricht:**

```
Betreff: Ihre Fotos sind fertig! 📸

Hallo [Name]!

Vielen Dank für das großartige Shooting! Ihre Fotos stehen jetzt zum Ansehen und Herunterladen bereit.

🔗 Link zur Galerie:
<Hier den Link zu Ihrer Galerie einfügen>

[Falls die Galerie ein Passwort hat:]
🔒 Passwort: sesja2024

💾 Sie können:
- Fotos online ansehen
- Einzelne Fotos herunterladen
- Die gesamte Galerie als ZIP herunterladen
- Den Link mit Familie und Freunden teilen

Die Fotos sind ein Jahr lang verfügbar.

Bei Fragen melden Sie sich gerne jederzeit!

Mit freundlichen Grüßen
[Ihr Name]
```

<Tip>
  **Professioneller Zusatz:** Fügen Sie eine kurze Anleitung bei, wie man die Fotos ansieht und herunterlädt. Ihre Kunden werden es zu schätzen wissen!
</Tip>

### SMS

Ideal für das schnelle Teilen.

**Beispielnachricht:**

```
Hallo! Ihre Fotos sind fertig:
<Hier den Link zu Ihrer Galerie einfügen>

Passwort: ania2024

Viele Grüße!
```

### Messenger / WhatsApp

**Beispielnachricht:**

```
Hallo! 👋

Ihre Fotos sind fertig! 🎉

Link: https://jan-kowalski.pixdeliver.com/wesele-nowak
Passwort: wesele2024

Die Fotos sind ein Jahr lang verfügbar 😊
```

<Warning>
  Vermeiden Sie es, Links öffentlich zu posten, wenn die Galerie private Fotos enthält. Versenden Sie Links über private Nachrichten.
</Warning>

## Den Link mit mehreren Personen teilen

Der Galerie-Link kann von einer unbegrenzten Anzahl von Personen geöffnet werden.

**Szenarien:**

- **Hochzeit** – senden Sie den Link an das Brautpaar, das ihn dann an die Gäste weitergeben kann
- **Firmenveranstaltung** – senden Sie den Link an den Organisator, der ihn an die Teilnehmer weiterleitet
- **Familienshooting** – die Organisatoren können den Link mit der ganzen Familie teilen

<Info>
  **Keine Limits:** Eine unbegrenzte Anzahl von Personen kann die Galerie öffnen und ansehen. Es gibt kein Limit für Aufrufe oder Downloads.
</Info>

## Was Kunden nach dem Öffnen des Links sehen

### Galerie ohne Passwort

1. Der Kunde klickt auf den Link
2. Er sieht sofort die Galerieseite mit:
   - Galeriename
   - Titelbild (sogenanntes Cover Photo)
   - Beschreibung (falls hinzugefügt)
   - Raster mit Foto-Miniaturansichten
   - Schaltfläche „Gesamte Galerie herunterladen“

### Galerie mit Passwort

1. Der Kunde klickt auf den Link
2. Er sieht einen Bildschirm mit der Passwortabfrage:
   - Feld zur Passworteingabe
   - Schaltfläche „Galerie öffnen“
3. Nach Eingabe des korrekten Passworts:
   - Die Galerie öffnet sich
   - Das Passwort wird im Browser gespeichert
   - Es muss nicht erneut eingegeben werden (solange der Browser nicht geschlossen wird)

## Den Link ändern

### Slug (URL-Adresse) ändern

Sie können den Slug der Galerie in den Einstellungen ändern, aber:

<Warning>
  **Achtung!** Nach dem Ändern des Slugs:

  - Funktioniert der alte Link sofort nicht mehr
  - Müssen Sie Ihren Kunden den neuen Link senden
  - Gibt es keine Weiterleitung vom alten Link auf den neuen
  - **Ändern Sie den Slug nur, bevor Sie die Galerie teilen!**
</Warning>

**Anleitung:**

1. Gehen Sie zu **Galerie-Einstellungen**
2. Ändern Sie das Feld **Slug**
3. Speichern Sie die Änderungen
4. Kopieren Sie den neuen Link
5. Senden Sie ihn erneut an Ihre Kunden

### Subdomain ändern

Das Ändern der Subdomain wirkt sich auf alle Ihre Galerien aus:

<Warning>
  **Achtung!** Nach dem Ändern der Subdomain:

  - Funktionieren alle alten Links mit Subdomain nicht mehr
  - Müssen Sie die Links bei allen Kunden aktualisieren
  - **Ändern Sie die Subdomain nicht, nachdem Sie die Links versendet haben!**
</Warning>

## Zugriff verfolgen

PixDeliver verfolgt automatisch, wer die Galerie wann besucht.

**Was erfasst wird:**

- Anzahl der Galerie-Aufrufe
- Eindeutige Besucher
- Datum und Uhrzeit des Zugriffs
- Zugriffsquelle (einfacher Link vs. Subdomain)
- Anzahl der Downloads (gesamte Galerie und einzelne Fotos)

**Statistiken ansehen:**

1. Öffnen Sie die Galerie im Bereich
2. Klicken Sie auf **Statistiken** oder **Statistiken ansehen**
3. Sie sehen alle Analysedaten

<Info>
  Gehen Sie zum Abschnitt [Galerie-Statistiken](/de/statistiken/dashboard), um mehr über die Analyse zu erfahren.
</Info>

## Link-Sicherheit

### Zufälligkeit des Slugs

Auch wenn Sie Ihren eigenen Slug wählen können, ist es ratsam, für mehr Sicherheit ein zufälliges Element hinzuzufügen:

**Beispiele:**

- ✅ `wesele-ania-xk4mp` (Slug + zufällige Zeichen)
- ✅ `sesja-kowalskich-2024-a8k` (Slug + Jahr + Code)
- ❌ `wesele-ania` (zu vorhersehbar)
- ❌ `test` (sehr allgemein)

<Tip>
  Wenn die Galerie sensible Fotos enthält, verwenden Sie einen schwer zu erratenden Slug + ein Passwort für maximale Sicherheit.
</Tip>

### HTTPS

Alle PixDeliver-Links verwenden eine verschlüsselte HTTPS-Verbindung:

- ✅ Sichere Datenübertragung
- ✅ Schutz vor Abhören
- ✅ SSL-Zertifikat

## Ablauf des Links

Der Link funktioniert in folgenden Situationen nicht mehr:

### 1. Die Galerie ist abgelaufen

Nach Ablauf eines Jahres ab Bezahlung der Galerie (oder nach Ablauf des Studio-Plans) und ohne Verlängerung sehen Kunden, die den Link öffnen, eine Seite mit dem Hinweis, dass die Galerie abgelaufen ist – mit der Bitte, den Fotografen zu kontaktieren.

**Lösung:** Verlängern Sie die Galerie ([Galerie-Verlängerung](/de/zahlungen/galerie-verlaengerung)) oder erneuern Sie den [Studio-Plan](/de/zahlungen/konto-abonnement).

### 2. Die Galerie wurde gelöscht

Wenn Sie die Galerie gelöscht haben, funktioniert der Link sofort nicht mehr.

### 3. Slug-Änderung

Nach dem Ändern des Slugs funktioniert der alte Link nicht mehr (ohne Weiterleitung).

## Tipps und bewährte Vorgehensweisen

✅ **Empfohlen:**

- Verwenden Sie den Link mit Subdomain für einen professionellen Auftritt
- Fügen Sie der Nachricht mit dem Link eine Anleitung bei
- Geben Sie das Passwort an (falls eingestellt)
- Informieren Sie Ihre Kunden über die Gültigkeit der Galerie (1 Jahr)
- Senden Sie den Link erst nach Aktivierung der Galerie
- Prüfen Sie den Link in der Vorschau, bevor Sie ihn versenden
- Bewahren Sie eine Kopie der versendeten Links auf (für sich selbst)
- Fügen Sie der Nachricht Emojis für einen besseren Eindruck hinzu 📸✨

❌ **Vermeiden Sie:**

- Den Slug zu ändern, nachdem Sie den Link an Kunden gesendet haben
- Links öffentlich zu posten (bei privaten Galerien)
- Den Link zu einer nicht bezahlten Galerie zu versenden
- Den Link ohne Passwort zu versenden (falls es erforderlich ist)
- Sehr kurze, vorhersehbare Slugs ohne Passwort zu verwenden
- Hinweise auf ablaufende Galerien zu ignorieren

## Fehlerbehebung

**Problem:** Der Kunde kann den Link nicht öffnen. **Lösung:**

1. Prüfen Sie, ob die Galerie aktiv (bezahlt) ist
2. Prüfen Sie, ob der Link korrekt kopiert wurde (ohne Abschneiden)
3. Vergewissern Sie sich, dass die Galerie nicht abgelaufen ist
4. Prüfen Sie, ob Sie den Slug nicht nach dem Versenden des Links geändert haben
5. Senden Sie den Link erneut (kopieren Sie einen frischen Link aus dem Bereich)

**Problem:** Der Kunde sagt, dass der Link ein Passwort verlangt, aber ich habe kein Passwort gesendet. **Lösung:**

- Prüfen Sie die Galerie-Einstellungen – ist ein Passwort eingestellt?
- Falls ja, senden Sie dem Kunden das Passwort
- Falls kein Passwort gewünscht ist, entfernen Sie es in den Galerie-Einstellungen

**Problem:** Der Link funktioniert, aber die Seite ist leer (keine Fotos). **Lösung:**

1. Prüfen Sie, ob die Fotos korrekt verarbeitet wurden
2. Sehen Sie nach, ob es einen Hinweis auf fehlgeschlagene Konvertierungen gibt
3. Aktualisieren Sie die Seite (Strg+F5)
4. Versuchen Sie es in einem anderen Browser
5. Prüfen Sie, ob die Fotos im Verwaltungsbereich sichtbar sind

**Problem:** Ich habe den Link gesendet, aber der Kunde hat ihn verloren. **Lösung:**

- Kopieren Sie den Link erneut aus dem Bereich
- Senden Sie ihn dem Kunden noch einmal
- Bewahren Sie Kopien der versendeten Links in Ihren E-Mails auf
