## Überblick

Nach dem Hochladen können Sie Ihre Fotos frei organisieren: die Anzeigereihenfolge ändern, das Titelbild der Galerie festlegen und nicht benötigte Fotos löschen. Eine gut organisierte Galerie wirkt professionell und erleichtert es Kunden, die Fotos zu durchsuchen.

---

## Reihenfolge der Fotos ändern

<Info>
  Das System zeigt die Fotos standardmäßig in der Reihenfolge des Hochladens an.
</Info>

### Methode Drag and Drop

**Anleitung:**

1. Öffnen Sie die Galerie im Verwaltungsbereich
2. Suchen Sie das Foto, das Sie verschieben möchten
3. Klicken Sie auf das Foto und halten Sie es gedrückt
4. Ziehen Sie es an die gewünschte Stelle
5. Lassen Sie das Foto los
6. Die Reihenfolge wird automatisch gespeichert

**Ergebnis:** Das Foto erscheint an der neuen Position. Die Kunden sehen es in derselben Reihenfolge.

<Tip>
  **Strategie für die Anordnung der Fotos:**

  - Beginnen Sie mit den besten Fotos (der erste Eindruck zählt!)
  - Ordnen Sie chronologisch an (das erzählt die Geschichte des Ereignisses)
  - Gruppieren Sie ähnliche Aufnahmen zusammen
</Tip>

## Titelbild festlegen

Das Titelbild (Cover Photo) ist das erste Foto, das Kunden sehen, wenn sie den Galerie-Link öffnen. Es wird auch als Miniaturansicht in den Galerie-Listen angezeigt.

### Auswahl des Titelbilds

**Anleitung:**

1. Öffnen Sie die Galerie im Verwaltungsbereich
2. Wechseln Sie zum Reiter „Titelbild"
3. Laden Sie ein Titelbild hoch, das ganz oben in der Galerie angezeigt wird

**Ergebnis:** Dieses Foto wird nun als erstes in der Galerie angezeigt.

<Tip>
  Auf dieselbe Weise aktualisieren Sie das Titelbild, indem Sie auf „Titelbild ändern" klicken.
</Tip>

## Fotos löschen

### Löschen eines einzelnen Fotos

**Anleitung:**

1. Öffnen Sie die Galerie im Verwaltungsbereich
2. Suchen Sie das zu löschende Foto
3. Bewegen Sie den Mauszeiger über das Foto
4. Klicken Sie auf das Symbol der **Dropdown-Liste** und anschließend auf die Schaltfläche **Löschen**
5. Bestätigen Sie das Löschen im Dialogfenster

**Ergebnis:** Das Foto wird dauerhaft in allen Versionen gelöscht (Original, Miniaturansicht, komprimierte Version).

<Warning>
  **Das Löschen ist unwiderruflich!** Ein einmal gelöschtes Foto kann nicht wiederhergestellt werden. Stellen Sie sicher, dass Sie eine Sicherungskopie auf Ihrer Festplatte haben, falls Sie sie noch benötigen könnten.
</Warning>

### Löschen mehrerer Fotos (Bulk Delete)

Wenn Sie mehrere oder einige Dutzend Fotos auf einmal löschen möchten:

**Anleitung:**

1. Klicken Sie in der Galerie-Ansicht auf die Schaltfläche **Fotos auswählen** oder auf die Auswahl-Schaltfläche
2. Markieren Sie die zu löschenden Fotos (klicken Sie auf jedes einzelne)
3. Sie können auch folgende Optionen verwenden:
   - **Alle auswählen** – markiert alle sichtbaren Fotos
   - **Alle abwählen** – hebt die Markierung aller auf
4. Klicken Sie auf die Schaltfläche **Markierte löschen** (erscheint, nachdem Fotos markiert wurden)
5. Bestätigen Sie das Löschen im Dialogfenster

**Ergebnis:** Alle markierten Fotos werden dauerhaft gelöscht.

<Tip>
  **Vor dem Massenlöschen:**

  - Überprüfen Sie die Auswahl zweimal
  - Stellen Sie sicher, dass Sie Sicherungskopien haben
</Tip>

## Galerie-Vorschau

Bevor Sie die Galerie für Kunden freigeben, sollten Sie sich ansehen, wie sie aus deren Perspektive aussieht.

**Anleitung:**

1. Klicken Sie in der Galerie-Ansicht auf die Schaltfläche **Galerie-Vorschau**
2. Es öffnet sich ein neuer Tab mit der Kundenansicht
3. Durchsuchen Sie die Galerie so, wie es Ihre Kunden tun werden
4. Überprüfen Sie:
   - Die Reihenfolge der Fotos
   - Das Titelbild
   - Die Ladegeschwindigkeit
   - Die Anzeige in der Lightbox
   - Die Responsivität (ändern Sie die Fenstergröße)

**Ergebnis:** Sie sehen die Galerie mit den Augen des Kunden und können vor der Veröffentlichung Korrekturen vornehmen.

<Info>
  Der Vorschaumodus zeigt die Galerie genau so, wie sie die Kunden sehen werden – einschließlich Passwort (falls eingerichtet) und allen Download-Funktionen.
</Info>

## Bewährte Vorgehensweisen

✅ **Empfohlen:**

- Legen Sie die Reihenfolge der Fotos vor der Aktivierung der Galerie fest
- Ordnen Sie die Fotos so an, dass sie eine Geschichte erzählen
- Löschen Sie unscharfe, sich wiederholende oder misslungene Fotos
- Legen Sie ein Titelbild fest, das die Foto-Auswahl repräsentiert
- Nutzen Sie die Vorschau vor der Veröffentlichung
- Halten Sie die Balance: nicht zu viele, nicht zu wenige Fotos

❌ **Vermeiden Sie:**

- Das Hinzufügen von Fotos in zufälliger Reihenfolge
- Das Platzieren schwächerer Fotos am Anfang
- Das Belassen misslungener Fotos „für alle Fälle"
- Das Erstellen von Galerien mit Hunderten sehr ähnlicher Fotos
- Das Festlegen eines abstrakten oder detailarmen Fotos als Titelbild
- Häufige Änderungen der Anordnung nach der Freigabe für Kunden

## Fehlerbehebung

**Problem:** Ich kann die Reihenfolge der Fotos nicht ändern. **Lösung:**

- Überprüfen Sie, ob Sie die Sortierung aktiviert haben
- Stellen Sie sicher, dass Sie das Foto ziehen und nicht nur darauf klicken
- Versuchen Sie, die Seite zu aktualisieren (Strg+F5)

**Problem:** Das Titelbild wird nicht aktualisiert. **Lösung:**

- Aktualisieren Sie die Browserseite
- Leeren Sie den Browser-Cache (Strg+Umschalt+Entf)
- Warten Sie einige Minuten, bis die Änderungen übernommen werden
- Stellen Sie sicher, dass das Foto korrekt verarbeitet wurde

**Problem:** Ich habe ein Foto versehentlich gelöscht. **Lösung:**

- Leider ist das Löschen unwiderruflich
- Sie können das Foto erneut aus einer Sicherungskopie hochladen
- Bewahren Sie die Originale immer auf einer lokalen Festplatte auf

**Problem:** Drag & Drop funktioniert nicht. **Lösung:**

- Überprüfen Sie, ob Sie einen unterstützten Browser verwenden (Chrome, Firefox, Edge, Safari)
- Aktualisieren Sie den Browser auf die neueste Version
- Deaktivieren Sie Browser-Erweiterungen, die Skripte blockieren könnten
