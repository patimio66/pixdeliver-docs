## Überblick

Nach dem Erstellen einer Galerie können Sie ihre Einstellungen jederzeit bearbeiten: Name, Slug (URL-Adresse), Beschreibung, Passwort und weitere Parameter ändern. Alle Änderungen sind sofort wirksam und für die Kunden unmittelbar nach dem Speichern sichtbar.

---

## Zugriff auf die Galerie-Einstellungen

**Anleitung:**

1. Melden Sie sich bei PixDeliver an
2. Wechseln Sie zu den **Galerien**
3. Suchen Sie die Galerie in der Liste und öffnen Sie sie
4. Öffnen Sie den Reiter **Einstellungen**
5. Es öffnet sich der Einstellungsbereich der Galerie

## Bearbeitung der grundlegenden Informationen

### Galeriename ändern

**Anleitung:**

1. Suchen Sie im Bereich **Grundlegendes** das Feld **Galeriename**
2. Geben Sie einen neuen Namen ein (bis zu 100 Zeichen)
3. Klicken Sie auf **Änderungen speichern**

**Ergebnis:** Der neue Name wird den Kunden auf der Galerie-Seite und in Ihrem Verwaltungsbereich angezeigt.

<Info>
  Der Galeriename ist für die Kunden sichtbar. Er sollte aussagekräftig und professionell sein.
</Info>

### Slug (URL-Adresse) ändern

**Anleitung:**

1. Suchen Sie im Bereich **Grundlegendes** das Feld **Slug**
2. Geben Sie einen neuen Slug ein (z. B. „hochzeit-kasia-und-mariusz-2024")
   - Nur Kleinbuchstaben, Ziffern und Bindestriche
   - Mindestens 3 Zeichen
   - Muss innerhalb Ihres Kontos eindeutig sein
3. Das System prüft die Verfügbarkeit in Echtzeit
4. Klicken Sie auf **Änderungen speichern**

**Ergebnis:** Die URL-Adresse der Galerie ändert sich. Alte Links funktionieren nicht mehr.

<Warning>
  **Achtung!** Nach dem Ändern des Slugs:

  - Der alte Link funktioniert sofort nicht mehr
  - Sie müssen den Kunden den neuen Link senden
  - Kunden, die den alten Link verwenden, sehen einen 404-Fehler
  - **<u>Ändern Sie den Slug nur, bevor Sie die Galerie für Kunden freigeben!</u>**
</Warning>

### Beschreibung bearbeiten

**Anleitung:**

1. Suchen Sie im Bereich **Grundlegendes** das Feld **Beschreibung**
2. Geben Sie eine Beschreibung der Galerie ein oder aktualisieren Sie sie
3. Sie können formatierten Text sowie Emojis verwenden
4. Klicken Sie auf **Änderungen speichern**

**Ergebnis:** Die Beschreibung wird den Kunden unter dem Galerienamen angezeigt.

<Tip>
  Neben dem Beschreibungsfeld finden Sie die Schaltfläche **Mit KI generieren** – sie erstellt 3 Beschreibungsvorschläge auf Basis des Galerienamens und eines kurzen Briefings. Wenn bereits eine Beschreibung vorhanden ist, ersetzt die Schaltfläche **Ersetzen** den aktuellen Text.
</Tip>

<Tip>
  **Ideen für Elemente der Beschreibung:**

  - „Laden Sie Ihre Lieblingsfotos herunter und teilen Sie sie mit Freunden!"
  - „Vielen Dank für einen wunderbaren Tag! ❤️"
  - „Fotos bis zum 31.12.2024 zum Herunterladen verfügbar"
</Tip>

## Passwortverwaltung

### Passwort zur Galerie hinzufügen

Wenn die Galerie ohne Passwort erstellt wurde, können Sie es jederzeit hinzufügen.

**Anleitung:**

1. Öffnen Sie die ausgewählte Galerie
1. Geben Sie im Reiter **Einstellungen** ein Passwort ein
4. Speichern Sie die Änderungen

**Ergebnis:** Ab jetzt müssen die Kunden vor dem Zugriff auf die Galerie ein Passwort eingeben.

### Passwort ändern

**Anleitung:**

1. Suchen Sie im Bereich **Sicherheit** das aktuelle Passwort (es ist sichtbar)
2. Geben Sie ein neues Passwort in das Feld **Passwort** ein
3. Bestätigen Sie das neue Passwort
4. Speichern Sie die Änderungen

**Ergebnis:** Das alte Passwort funktioniert nicht mehr. Die Kunden müssen das neue Passwort verwenden.

<Warning>
  Wenn Sie das Passwort nach der Freigabe der Galerie ändern, müssen Sie die Kunden über das neue Passwort informieren. Andernfalls verlieren sie den Zugang zur Galerie.
</Warning>

### Passwort entfernen

Wenn Sie die Galerie ohne Passwort öffentlich zugänglich machen möchten:

**Anleitung:**

1. Leeren Sie im Bereich **Einstellungen** das Feld **Passwortschutz**
2. Speichern Sie die Änderungen

**Ergebnis:** Die Galerie wird öffentlich zugänglich. Jeder mit dem Link kann sie sehen.

## Lieferung und Verkauf

Im Bereich **Lieferung und Verkauf** legen Sie fest, ob die Galerie kostenlos oder kostenpflichtig ist:

- **Kostenlos** – die Kunden laden die Originale sofort herunter
- **Kostenpflichtig – Kauf der gesamten Galerie** – die Kunden kaufen die gesamte Galerie
- **Kostenpflichtig – Kauf einzelner Fotos** – die Kunden kaufen ausgewählte Fotos

In den kostenpflichtigen Modi legen Sie hier die Preisliste fest (**Grundpreis**, **Preis pro Foto**, **inbegriffene Fotos**) sowie den **Text des Wasserzeichens**.

<Warning>
  Der Liefermodus ist **gesperrt**, wenn in der Galerie eine aktive Kaufanfrage vorliegt. Rechnen Sie die Anfrage zunächst ab oder stornieren Sie sie im Reiter **Kaufanfragen**.
</Warning>

<Info>
  Vollständige Leitfäden: [Verkauf von Galerien](/de/lieferung-und-verkauf/galerien-verkaufen) sowie [Kaufanfragen und Abrechnung](/de/lieferung-und-verkauf/anfragen-und-abrechnung).
</Info>

## Galerie-URL

Jede Galerie hat zwei mögliche URL-Adressen – aber pro Konto kann nur eine Art verwendet werden:

### 1. Standardadresse (Root-Domain)

```
https://pixdeliver.com/g/[slug]
```

Beispiel: `https://pixdeliver.com/g/hochzeit-jan-und-ania`

### 2. Adresse mit Subdomain (falls eingerichtet)

```
https://[ihre-subdomain].pixdeliver.com/[slug]
```

Beispiel: `https://jan-kowalski.pixdeliver.com/hochzeit-jan-und-ania`

<Tip>
  **Eine Subdomain ist professioneller:**

  - Sie enthält Ihren Namen oder Firmennamen
  - Sie stärkt die Wiedererkennung Ihrer Marke
  - Sie wirkt persönlicher
</Tip>

### Link kopieren

**Sie können dies auf verschiedene Weise tun:**

- Suchen Sie in den Galerie-Einstellungen den Bereich **Öffentliche Adresse** und übertragen Sie sie
- Klicken Sie auf jeder Unterseite der ausgewählten Galerie auf die Schaltfläche **Öffentlichen Link kopieren**

**Ergebnis:** Der Link wird in die Zwischenablage kopiert und Sie können ihn an die Kunden senden.

## Status und Ablauf

### Mögliche Galerie-Status

- **Nicht bezahlt** – Die Galerie ist nicht aktiv, öffentlich nicht verfügbar
- **Aktiv** – Die Galerie ist bezahlt und für Kunden verfügbar
- **Abgelaufen** – Die Galerie hat ihre Gültigkeit beendet (nach 1 Jahr)

### Ablaufdatum

Für aktive Galerien sehen Sie das Ablaufdatum auf der Seite mit der Galerie-Liste:

**Beispiel:**

```
Erstellt: 15.06.2025
Läuft ab: 15.06.2026
```

<Info>
  Das System sendet automatische Benachrichtigungen:

  - **30 Tage vor dem Ablauf** – ab diesem Zeitpunkt können Sie die Galerie um ein Jahr verlängern
  - **14 Tage vor dem Ablauf** – Erinnerung
  - **7 Tage vor dem Ablauf** – Erinnerung
</Info>

### Galerie verlängern

**Anleitung:**

1. Öffnen Sie die ausgewählte Galerie – Sie sehen eine Benachrichtigung über den Ablauf
2. Der Plan bleibt derselbe wie bei der Erstellung der Galerie
3. Wechseln Sie zur Zahlung
4. Nach der Bezahlung ist die Galerie für ein weiteres Jahr aktiv

**Ergebnis:** Das Ablaufdatum wird um zusätzlich 1 Jahr verlängert.

## Galerie löschen

<Danger>
  Stellen Sie vor dem Löschen der Galerie sicher, dass Sie Sicherungskopien aller Fotos haben!
</Danger>

### Dauerhaftes Löschen der Galerie

Wenn Sie die Galerie nicht mehr benötigen:

**Anleitung:**

1. Scrollen Sie zum Ende der Galerie-Einstellungen
2. Suchen Sie den Bereich **Gefahrenzone**
3. Klicken Sie auf die Schaltfläche **Galerie löschen**
4. Lesen Sie die Warnung
5. Geben Sie zur Bestätigung den Namen der Galerie ein
6. Klicken Sie auf **Galerie dauerhaft löschen**

**Ergebnis:** Die Galerie und alle Fotos werden dauerhaft gelöscht.

<Warning>
  **Diese Aktion ist unwiderruflich!**

  - Alle Fotos der ausgewählten Galerie werden gelöscht
  - Der Galerie-Link funktioniert nicht mehr
  - Die Statistiken gehen verloren
  - Die Fotoauswahlen werden gelöscht
  - **Eine Wiederherstellung der Daten ist nicht möglich**
</Warning>

## Datenexport

### Alle Fotos herunterladen

Vor dem Löschen der Galerie oder zur Archivierung können Sie alle Fotos herunterladen:

**Anleitung:**

1. Öffnen Sie die ausgewählte Galerie
1. Klicken Sie im Galerie-Menü auf die Schaltfläche **Herunterladen**

**Ergebnis:** Sie laden eine ZIP-Datei mit allen Fotos der Galerie herunter.

## Tipps und bewährte Vorgehensweisen

✅ **Empfohlen:**

- Legen Sie einen aussagekräftigen Galerienamen fest
- Fügen Sie eine Beschreibung mit Anweisungen für die Kunden hinzu
- Verwenden Sie eine Subdomain für einen professionellen Auftritt
- Ändern Sie den Slug nicht nach der Freigabe der Galerie
- Überprüfen Sie regelmäßig die Ablaufdaten der Galerien
- Laden Sie vor dem Löschen der Galerie eine Sicherungskopie herunter
- Richten Sie für sensible Sessions ein Passwort ein
- Verlängern Sie Galerien im Voraus (warten Sie nicht bis zum letzten Tag)

❌ **Vermeiden Sie:**

- Das Ändern des Slugs, nachdem Sie den Link an Kunden gesendet haben
- Das Löschen von Galerien ohne vorheriges Herunterladen einer Sicherungskopie
- Dasselbe Passwort für alle Galerien zu verwenden
- Galerien ohne Namen oder mit dem Namen „Test" zu belassen

## Fehlerbehebung

**Problem:** Ich kann den Slug nicht ändern – er ist vergeben. **Lösung:**

- Der Slug muss innerhalb Ihres Kontos eindeutig sein
- Überprüfen Sie, ob Sie nicht bereits eine Galerie mit demselben Slug haben
- Fügen Sie Ziffern, ein Datum oder eine zusätzliche Beschreibung hinzu (z. B. `-2024`, `-hochzeit`)

**Problem:** Die Kunden können die Galerie nach dem Ändern des Slugs nicht öffnen. **Lösung:**

- Senden Sie ihnen den neuen Link mit dem aktualisierten Slug
- Der alte Link funktioniert nicht mehr und es gibt keine Weiterleitung
- Ändern Sie den Slug künftig <u>nicht</u> nach der Freigabe der Galerie

**Problem:** Das Passwort funktioniert für die Kunden nicht. **Lösung:**

- Überprüfen Sie, ob das Passwort korrekt eingegeben ist (Groß- und Kleinschreibung ist relevant)
- Stellen Sie sicher, dass das Passwort keine Leerzeichen am Anfang oder Ende enthält
- Sehen Sie sich das aktuelle Passwort in den Galerie-Einstellungen an
- Senden Sie den Kunden das Passwort erneut (kopieren Sie es direkt aus den Einstellungen)

**Problem:** Die Galerie ist abgelaufen, aber ich möchte sie nicht löschen. **Lösung:**

- Verlängern Sie die Galerie innerhalb von 14 Tagen nach dem Ablauf
- Nach 14 Tagen kann die Galerie automatisch gelöscht werden
- Die Verlängerung stellt die Verfügbarkeit der Galerie für ein weiteres Jahr wieder her

**Problem:** Ich sehe die Schaltfläche „Änderungen speichern" nicht. **Lösung:**

- Scrollen Sie das Formular bis zum Ende – die Schaltfläche befindet sich unten
- Aktualisieren Sie die Seite (Strg+F5)
- Überprüfen Sie, ob Sie alle erforderlichen Felder ausgefüllt haben
