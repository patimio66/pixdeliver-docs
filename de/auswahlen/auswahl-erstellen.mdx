## Überblick

Auswahlformulare ermöglichen es Kunden, ihre Lieblingsfotos direkt aus der Galerie auszuwählen. Das ist die ideale Lösung, wenn der Kunde eine bestimmte Anzahl von Fotos auswählen muss (z. B. für ein Album, Rahmen, Druck) oder seine Lieblingsaufnahmen kennzeichnen möchte.

**Wichtige Funktionen:**
- Kunden wählen Fotos durch eine einfache Auswahl (Klick)
- Sie können Titel und Beschreibung des Formulars festlegen
- Optionales Ablaufdatum der Auswahl
- Erfassung von Name und E-Mail des Kunden
- Sofortige Benachrichtigung über eine neue Einreichung
- Vollständige Vorschau der ausgewählten Fotos

---

## Wann sollten Sie Fotoauswahlen verwenden?

### Ideale Einsatzszenarien:

**Auswahl für ein Album:**
- Der Kunde hat 300 Fotos von der Hochzeit, aber das Album fasst nur 80
- „Wählen Sie 80 Lieblingsfotos für das Hochzeitsalbum aus"

**Auswahl für den Druck:**
- Familiensession - der Kunde möchte 10 Fotos für Rahmen auswählen
- „Wählen Sie 10 Fotos für den Druck im Format 30×40 cm aus"

**Favorit des Kunden:**
- Sie möchten wissen, welche Fotos dem Kunden am besten gefallen haben
- „Wählen Sie Ihre 20 Lieblingsfotos aus der Session aus"

**Fotos zur Retusche:**
- Sie bieten eine fortgeschrittene Retusche einiger Fotos an
- „Wählen Sie 5 Fotos für die professionelle Retusche aus"

**Veröffentlichung im Portfolio:**
- Sie bitten den Kunden um Zustimmung zur Veröffentlichung bestimmter Fotos
- „Markieren Sie die Fotos, die ich im Portfolio veröffentlichen darf"

## Schritt für Schritt

### 1. Öffnen Sie die Galerie

**Anleitung:**
1. Melden Sie sich bei PixDeliver an
2. Wechseln Sie zum **Dashboard**
3. Öffnen Sie die Galerie, für die Sie eine Auswahlanfrage erstellen möchten
4. Die Galerie muss aktiv (bezahlt) sein

**Ergebnis:** Sie befinden sich in der Galerieansicht.

---

### 2. Wechseln Sie zum Bereich Auswahlanfragen

**Anleitung:**
1. Suchen Sie im Galeriemenü den Tab **Auswahlanfragen**
2. Klicken Sie, um zur Liste der Auswahlanfragen zu gelangen
3. Sie sehen eine Liste der bestehenden Auswahlanfragen (sofern vorhanden) oder einen Button zum Erstellen einer neuen

**Ergebnis:** Sie befinden sich im Bereich zur Verwaltung der Auswahlanfragen.

---

### 3. Erstellen Sie eine neue Auswahlanfrage

**Anleitung:**
1. Klicken Sie auf den Button **Auswahlanfrage erstellen** oder **Neue Auswahlanfrage**
2. Es öffnet sich das Formular zum Erstellen einer Auswahlanfrage

**Ergebnis:** Sie sehen ein Formular mit auszufüllenden Feldern.

---

### 4. Füllen Sie die Details der Auswahlanfrage aus

**Anleitung:**
1. **Titel der Auswahl** - Geben Sie einen aussagekräftigen Namen ein
   - Beispiel: „Fotoauswahl für das Hochzeitsalbum"
   - Beispiel: „Ihre 10 Lieblingsfotos"
   - Er ist für den Kunden sichtbar

2. **Beschreibung** - Fügen Sie Anweisungen für den Kunden hinzu
   - Beispiel: „Wählen Sie maximal 80 Fotos aus, die Sie im Hochzeitsalbum haben möchten"
   - Beispiel: „Markieren Sie 10 Fotos, die ich im Format 30×40 cm drucke"
   - Erklären Sie, was der Kunde tun soll

3. **Ablaufdatum** (optional)
   - Legen Sie eine Frist fest, bis wann der Kunde die Fotos auswählen soll
   - Beispiel: 2 Wochen ab heute
   - Lassen Sie das Feld leer, wenn es keine Frist gibt

4. **Status** - Markieren Sie als **Aktiv**
   - Aktiv = Kunden können das Formular ausfüllen
   - Inaktiv = das Formular ist deaktiviert (später verwendet)

5. Klicken Sie auf **Auswahlanfrage erstellen**

**Ergebnis:** Die Auswahlanfrage wird erstellt und Sie erhalten einen eindeutigen Link dazu.

<Tip>
**Eine gute Auswahlbeschreibung:**
- Geben Sie die Anzahl der Fotos klar an (z. B. „Wählen Sie maximal 10 Fotos aus")
- Erläutern Sie den Zweck (für Album, Druck, Retusche)
- Fügen Sie eine Frist hinzu (falls relevant): „Bitte bis zum 15. Juni auswählen"
- Verwenden Sie einen freundlichen Ton: „Wählen Sie Ihre Lieblingsfotos aus 💕"
</Tip>

## Beispiele für Auswahlen

### Beispiel 1: Hochzeitsalbum

```
Titel: Fotoauswahl für das Hochzeitsalbum

Beschreibung:
Hallo, liebe Anna und lieber Tomek!

Es ist Zeit, die Fotos für euer Hochzeitsalbum auszuwählen 💍

Das Album fasst 80 Fotos. Wählt eure schönsten Momente dieses besonderen Tages aus!

Auswahlfrist: 31. Mai 2024

Nachdem ihr die Fotos markiert habt, gebt eure Namen ein und klickt auf „Auswahl senden".

Ablaufdatum: 2024-05-31
Status: Aktiv
```

### Beispiel 2: Auswahl zur Retusche

```
Titel: Fotos für die fortgeschrittene Retusche

Beschreibung:
Hallo Kasia!

Wähle 5 Fotos aus, die wir einer professionellen Retusche unterziehen (Hautglättung, Farbkorrektur, Entfernung von Unreinheiten).

Markiere deine Favoriten und sende das Formular ab.

Die retuschierten Fotos erhältst du innerhalb von 5 Werktagen.

Ablaufdatum: Keines
Status: Aktiv
```

### Beispiel 3: Zustimmung zur Veröffentlichung

```
Titel: Zustimmung zur Veröffentlichung im Portfolio

Beschreibung:
Guten Tag!

Unsere Zusammenarbeit lag mir sehr am Herzen, und ich bin stolz auf die entstandenen Fotos.

Dürfte ich einige davon in meinem Portfolio veröffentlichen? Markieren Sie die Fotos, deren Veröffentlichung Sie zustimmen.

Wenn Sie natürlich lieber die volle Privatsphäre aller Fotos wahren möchten, markieren Sie einfach keines - das verstehe ich vollkommen!

Vielen Dank!

Ablaufdatum: Keines
Status: Aktiv
```

## Einstellungen der Auswahl

### Titel der Auswahl

- **Länge:** 5-100 Zeichen
- **Inhalt:** Aussagekräftiger, für den Kunden sichtbarer Name
- **Beispiele:** „Auswahl für das Album", „Ihre Top 10", „Fotos zum Druck"

### Beschreibung der Auswahl

- **Länge:** Bis zu 1000 Zeichen
- **Inhalt:** Detaillierte Anweisungen, Informationen zum Zweck und zur Frist
- **Formatierung:** Verwenden Sie Leerzeilen für eine bessere Lesbarkeit (Enter = neue Zeile)
- **Ton:** Freundlich, ermutigend, klar

### Ablaufdatum

**Optionale Funktion - wann verwenden:**
- ✅ Wenn Sie eine Frist für die Albumproduktion haben
- ✅ Wenn Sie den Kunden zu einer schnelleren Entscheidung motivieren möchten
- ✅ Wenn die Leistung zeitlich begrenzt ist

**Wann nicht festlegen:**
- ❌ Wenn der Kunde mehr Zeit zum Überlegen braucht
- ❌ Wenn es keine konkrete Frist gibt
- ❌ Für Archivgalerien

**Was nach dem Ablauf passiert:**
- Das Formular wird unzugänglich
- Der Kunde sieht die Meldung: „Diese Auswahl ist abgelaufen"
- Sie können das Ablaufdatum in den Einstellungen verlängern

### Status aktiv/inaktiv

**Aktiv:**
- Kunden können das Formular ausfüllen
- Der Link zur Auswahl funktioniert
- Es werden neue Einreichungen erfasst

**Inaktiv:**
- Das Formular ist deaktiviert
- Der Link führt zur Meldung „Auswahl inaktiv"
- Es kann keine neue Auswahl gesendet werden
- Nützlich, wenn Sie bereits alle benötigten Einreichungen gesammelt haben

## Auswahlanfrage bearbeiten

Sie können eine Auswahlanfrage jederzeit bearbeiten.

**Anleitung:**
1. Wechseln Sie in der Galerie zum Bereich **Auswahlanfragen**
2. Suchen Sie die Auswahlanfrage in der Liste
3. Klicken Sie auf **Bearbeiten** oder das Stiftsymbol
4. Aktualisieren Sie die gewünschten Felder:
   - Titel
   - Beschreibung
   - Ablaufdatum
   - Status
5. Klicken Sie auf **Änderungen speichern**

**Ergebnis:** Die Änderungen sind sofort wirksam. Die Kunden sehen die aktualisierten Informationen.

<Warning>
**Hinweis:** Wenn Sie die Beschreibung ändern, nachdem einige Kunden das Formular bereits ausgefüllt haben, werden diese nicht automatisch über die Änderungen benachrichtigt.
</Warning>

---

## Auswahlanfrage löschen

Sie können eine Auswahlanfrage zusammen mit allen Einreichungen löschen.

**Anleitung:**
1. Wechseln Sie zum Bereich **Auswahlanfragen**
2. Suchen Sie die zu löschende Auswahlanfrage
3. Klicken Sie auf **Löschen** oder das Papierkorbsymbol
4. Bestätigen Sie das Löschen

**Ergebnis:** Die Auswahlanfrage und alle Einreichungen werden dauerhaft gelöscht.

<Warning>
**Diese Aktion ist unumkehrbar!** Laden Sie vor dem Löschen die Einreichungen herunter oder machen Sie einen Screenshot, falls Sie die Daten behalten möchten.
</Warning>

## Mehrere Auswahlanfragen für eine Galerie

Sie können mehrere Auswahlanfragen für dieselbe Galerie erstellen.

**Beispielszenarien:**

**Hochzeit - verschiedene Zwecke:**
1. „Auswahl für das Album" - 80 Fotos
2. „Fotos für die Eltern" - 20 Fotos für die Eltern der Braut
3. „Diashow zum Jahrestag" - 30 Fotos für die Präsentation

**Familiensession - verschiedene Personen:**
1. „Auswahl der Mutter" - die Mutter wählt ihre Favoriten
2. „Auswahl des Vaters" - der Vater wählt unabhängig
3. „Gemeinsame Auswahl" - die endgültige Auswahl nach der Besprechung

**Verschiedene Phasen:**
1. „Erste Auswahl - breit" - Auswahl von 100 Fotos aus 500
2. „Zweite Auswahl - eng" - Auswahl der endgültigen 50 aus diesen 100

## Tipps und Best Practices

✅ **Empfohlen:**
- Verwenden Sie klare, konkrete Anweisungen („Wählen Sie 10 Fotos aus", nicht „Wählen Sie ein paar aus")
- Erläutern Sie den Zweck der Auswahl (wofür die ausgewählten Fotos verwendet werden)
- Fügen Sie eine Frist hinzu, falls relevant
- Verwenden Sie eine emotionale, ermutigende Sprache
- Testen Sie das Formular selbst, bevor Sie es an den Kunden senden
- Setzen Sie das Ablaufdatum mit Puffer (der Kunde braucht vielleicht Zeit)
- Erstellen Sie separate Auswahlanfragen für verschiedene Zwecke
- Aktivieren Sie eine Auswahlanfrage erst, wenn die Galerie fertig ist

❌ **Vermeiden Sie:**
- Unklare Anweisungen („Wählen Sie die Fotos aus, die Ihnen gefallen")
- Zu kurze Fristen (der Kunde schafft es womöglich nicht)
- Das Erstellen von Auswahlanfragen vor dem Hochladen aller Fotos
- Das Ändern der Regeln nach dem Versand des Links an die Kunden
- Zu lange Beschreibungen (der Kunde liest sie nicht)
- Die Verwendung von technischer oder Fachsprache

## Fehlerbehebung

**Problem:** Ich sehe den Button „Auswahlanfrage erstellen" nicht
**Lösung:**
- Prüfen Sie, ob die Galerie aktiv (bezahlt) ist
- Auswahlanfragen funktionieren nur für aktive Galerien
- Bezahlen Sie die Galerie, um Auswahlanfragen freizuschalten

**Problem:** Der Kunde sagt, er könne die Auswahl nicht senden
**Lösung:**
1. Prüfen Sie den Status der Auswahl - ist sie aktiv?
2. Prüfen Sie das Ablaufdatum - ist es nicht überschritten?
3. Bitten Sie den Kunden um einen Screenshot des Fehlers
4. Prüfen Sie, ob der Kunde Fotos markiert hat
5. Stellen Sie sicher, dass der Kunde Name und E-Mail ausgefüllt hat

**Problem:** Ich habe eine Auswahlanfrage erstellt, weiß aber nicht, wie ich den Link versende
**Lösung:**
- Siehe nächster Abschnitt: [Auswahl teilen](/de/auswahlen/auswahl-teilen)
- Der Link zur Auswahl wird nach ihrer Erstellung automatisch generiert

**Problem:** Ich möchte die Beschreibung ändern, nachdem ich den Link an die Kunden gesendet habe
**Lösung:**
- Sie können die Beschreibung jederzeit bearbeiten
- Die Änderungen sind sofort wirksam
- Kunden, die das Formular noch nicht ausgefüllt haben, sehen die neue Beschreibung
- Diejenigen, die es bereits gesendet haben, werden nicht über die Änderung benachrichtigt

## Häufig gestellte Fragen

**Kann ich die Anzahl der markierten Fotos begrenzen?**
Derzeit gibt es kein technisches Limit. Sie müssen den Kunden in der Beschreibung darum bitten (z. B. „Wählen Sie maximal 10 Fotos aus").

**Kann ein Kunde das Formular mehrmals ausfüllen?**
Ja, ein Kunde kann mehrere Einreichungen senden. Sie sehen alle im Bereich zur Verwaltung der Einreichungen.

**Funktioniert eine Auswahlanfrage für eine Galerie mit Passwort?**
Ja! Der Kunde gibt zunächst das Passwort zur Galerie ein und kann dann das Auswahlformular öffnen.

**Kann ich die Auswahl des Kunden vor dem offiziellen Senden sehen?**
Nein. Der Kunde muss auf „Auswahl senden" klicken, damit seine Auswahl bei Ihnen ankommt.

**Wie lange werden die Daten gespeichert?**
Die Einreichungen werden für die gesamte Gültigkeitsdauer der Galerie (1 Jahr) oder bis zum Löschen der Auswahl/Galerie gespeichert.
