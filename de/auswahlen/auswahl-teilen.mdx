## Überblick

Nach dem Erstellen eines Auswahlformulars erhalten Sie einen eindeutigen Link, den Sie an Ihre Kunden senden können. Die Kunden öffnen diesen Link, markieren ihre Lieblingsfotos direkt in der Galerie und senden ihre Auswahl. Sie erhalten eine sofortige Benachrichtigung über jede neue Einreichung.

---

## Format des Auswahllinks

Jede Auswahlanfrage hat ihren eigenen eindeutigen Link, der aus einem Token (16 Zeichen) besteht.

### Basislink

```
https://pixdeliver.com/select/[16-zeichen-token]
```

**Beispiel:**
```
https://pixdeliver.com/select/a8k3mp9xq2n4r7v1
```

### Link mit Subdomain

Wenn Sie eine Subdomain konfiguriert haben:

```
https://[ihre-subdomain].pixdeliver.com/select/[token]
```

**Beispiel:**
```
https://jan-kowalski.pixdeliver.com/select/a8k3mp9xq2n4r7v1
```

<Info>
Beide Links führen zum selben Auswahlformular. Sie können einen beliebigen verwenden - die Subdomain wirkt professioneller.
</Info>

## Auswahllink kopieren

**Anleitung:**
1. Wechseln Sie in der Galerie zum Bereich **Auswahlanfragen**
2. Suchen Sie die erstellte Auswahlanfrage in der Liste
3. Klicken Sie auf den Button **Link kopieren** oder das Ketten-/Linksymbol
4. Der Link wird in die Zwischenablage kopiert

**Alternativ:**
1. Klicken Sie auf den Namen der Auswahlanfrage, um die Details zu öffnen
2. Im Abschnitt **Link zum Teilen** finden Sie den Link
3. Klicken Sie auf **Link kopieren**

**Ergebnis:** Der Link ist bereit zum Versand an die Kunden.

## Anweisungen für die Kunden

### Grundanleitung (zum Einfügen in die E-Mail)

```
So wählen Sie Fotos aus:

1. Klicken Sie auf den Link zum Auswahlformular
2. Wenn die Galerie ein Passwort hat, geben Sie es ein (Sie finden es in der vorherigen E-Mail)
3. Sie sehen alle Fotos der Galerie
4. Klicken Sie auf jedes Foto, das Sie auswählen möchten - es erscheint ein ✓
5. Sie können Fotos beliebig markieren und die Markierung aufheben
6. Nachdem Sie die passende Anzahl an Fotos ausgewählt haben:
   - Geben Sie Ihren Namen in das Feld „Name" ein
   - Geben Sie Ihre E-Mail in das Feld „E-Mail" ein
   - Klicken Sie auf „Auswahl senden"
7. Fertig! Sie erhalten eine Bestätigung per E-Mail

💡 Tipp: Sie können ein Foto in voller Größe öffnen, indem Sie darauf klicken.
```

### Erweiterte Anleitung mit technischer Hilfe

```
So wählen Sie Fotos aus - Schritt für Schritt:

SCHRITT 1: Öffnen Sie das Formular
• Klicken Sie auf den Link, den Sie per E-Mail/SMS erhalten haben
• Wenn die Galerie passwortgeschützt ist, geben Sie das Passwort ein

SCHRITT 2: Durchsehen und markieren
• Sie sehen alle Fotos der Galerie in einem Raster
• Klicken Sie auf ein Foto, um es zu markieren (es erscheint ein ✓)
• Klicken Sie erneut, um die Markierung aufzuheben
• Sie können Ihre Auswahl beliebig ändern

SCHRITT 3: Auswahl prüfen
• Oben auf dem Bildschirm sehen Sie die Anzahl der markierten Fotos
• Scrollen Sie bis zum Ende der Galerie nach unten

SCHRITT 4: Senden Sie das Formular
• Geben Sie Ihren Namen ein
• Geben Sie Ihre E-Mail ein (Sie erhalten eine Bestätigung)
• Klicken Sie auf „Auswahl senden"
• Sie sehen eine Bestätigungsmeldung

HILFE:
• Ich weiß nicht, wie viele Fotos ich markiert habe: Prüfen Sie den Zähler oben
• Ich möchte die Auswahl nach dem Senden ändern: Füllen Sie das Formular erneut aus
• Das Formular funktioniert nicht: Versuchen Sie es in einem anderen Browser (Chrome, Firefox)
• Andere Probleme: Schreiben Sie mir!
```

## Auswahl-Token

### Sicherheit des Tokens

Das Auswahl-Token ist eine 16-stellige zufällige Zeichenfolge:
- Sehr schwer zu erraten
- Eindeutig für jede Auswahl
- Funktioniert wie ein Zugangspasswort zum Formular

**Beispiele für Tokens:**
- `a8k3mp9xq2n4r7v1`
- `z5t2qw8r3n6m9k4p`
- `h7j2k9m3x8q5w1n4`

### Wer kann das Formular öffnen?

- Jeder, der den Link (das Token) hat
- Es gibt kein zusätzliches Passwort (es sei denn, die Galerie hat ein Passwort)
- Der Link kann an mehrere Personen weitergegeben werden

<Info>
**Galerie mit Passwort:** Wenn die Galerie passwortgeschützt ist, muss der Kunde zunächst das Passwort zur Galerie eingeben und kann dann das Auswahlformular öffnen.
</Info>

## Mehrere Personen, ein Formular

Sie können denselben Auswahllink an mehrere Personen senden.

**Szenarien:**

**Hochzeit - das Brautpaar:**
- Senden Sie den Link an die Braut und den Bräutigam
- Sie können die Fotos gemeinsam oder getrennt auswählen
- Jeder kann seine eigene Auswahl senden
- Sie sehen alle Einreichungen

**Familie - jeder wählt einzeln:**
- Die Mutter wählt ihre Favoriten
- Der Vater wählt seine Favoriten
- Die Kinder wählen ihre Favoriten
- Sie vergleichen die Auswahlen und erstellen die endgültige Liste

**Event - Teilnehmer wählen Fotos von sich:**
- Senden Sie den Link an alle Teilnehmer
- Jeder kann die Fotos markieren, auf denen er zu sehen ist
- Alle füllen dasselbe Formular aus

## Status verfolgen

### Prüfen, wer seine Auswahl gesendet hat

**Anleitung:**
1. Wechseln Sie in der Galerie zum Bereich **Auswahlanfragen**
2. Suchen Sie die Auswahlanfrage in der Liste
3. Sie sehen die Anzahl der Einreichungen: „Einreichungen: 3"
4. Klicken Sie auf die Auswahlanfrage, um die Details zu sehen
5. Liste aller Einreichungen mit Namen und Daten

**Ergebnis:** Sie wissen, wer bereits seine Auswahl gesendet hat und wer noch nicht.

### E-Mail-Benachrichtigungen

Sie erhalten eine automatische Benachrichtigung, wenn ein Kunde seine Auswahl sendet:

**Betreff:** „Neue Fotoauswahl von [Name des Kunden]"

**Inhalt:**
- Name und E-Mail des Kunden
- Name der Galerie
- Titel der Auswahl
- Anzahl der ausgewählten Fotos
- Datum und Uhrzeit der Einreichung
- Link zur Vorschau der Auswahl

## Tipps und Best Practices

✅ **Empfohlen:**
- Senden Sie den Link zusammen mit einer klaren Anleitung
- Kombinieren Sie den Galerielink und den Auswahllink in einer E-Mail
- Geben Sie eine Deadline an, falls relevant
- Erinnern Sie die Kunden frühzeitig (eine Woche, einen Tag vorher)
- Verwenden Sie eine emotionale, ermutigende Sprache
- Fügen Sie Ihre Kontaktdaten für Fragen/Probleme hinzu
- Testen Sie den Link selbst vor dem Versand
- Verwenden Sie den Link mit Subdomain für mehr Professionalität

❌ **Vermeiden Sie:**
- Den Link ohne Anleitung zu versenden
- Zu kurze Deadlines (der Kunde braucht Zeit!)
- Unklare Mitteilungen („Wählen Sie irgendwelche Fotos aus")
- Den Link zu einer inaktiven Auswahl zu senden
- Das Ändern der Regeln nach dem Versand des Links (Änderung der Fotoanzahl usw.)
- Den Link öffentlich zu veröffentlichen (wenn die Galerie privat ist)

## Fehlerbehebung

**Problem:** Der Kunde sagt, der Link funktioniere nicht
**Lösung:**
1. Prüfen Sie den Status der Auswahl - ist sie aktiv?
2. Prüfen Sie das Ablaufdatum - ist es nicht überschritten?
3. Stellen Sie sicher, dass der Link vollständig kopiert wurde (nicht abgeschnitten)
4. Senden Sie den Link erneut (kopieren Sie einen frischen)
5. Bitten Sie den Kunden um einen Screenshot des Fehlers

**Problem:** Der Kunde weiß nicht, wie er Fotos markiert
**Lösung:**
- Senden Sie eine ausführliche Schritt-für-Schritt-Anleitung
- Bieten Sie ein Telefon- oder Videogespräch an, um den Vorgang zu zeigen
- Stellen Sie sicher, dass der Kunde einen Computer/Tablet verwendet (auf dem Smartphone ist es womöglich weniger intuitiv)

**Problem:** Der Kunde hat seine Auswahl gesendet, möchte sie aber ändern
**Lösung:**
- Der Kunde kann das Formular erneut ausfüllen
- Sie sehen beide Einreichungen
- Verwenden Sie die letzte Einreichung oder fragen Sie den Kunden, welche Auswahl die endgültige ist

**Problem:** Ich habe vergessen, dem Kunden den Link zu senden
**Lösung:**
- Wechseln Sie zum Bereich Auswahlanfragen
- Kopieren Sie den Link erneut
- Senden Sie ihn dem Kunden mit einer Entschuldigung 😊

**Problem:** Der Auswahllink verlangt ein Galeriepasswort
**Lösung:**
- Das ist normal, wenn die Galerie passwortgeschützt ist
- Stellen Sie sicher, dass Sie dem Kunden das Galeriepasswort gesendet haben
- Der Kunde muss zuerst die Galerie öffnen (Passwort eingeben) und kann dann das Auswahlformular nutzen

## Häufig gestellte Fragen

**Kann ich den Link an mehrere Personen senden?**
Ja! Derselbe Link kann von mehreren Personen verwendet werden. Jede kann das Formular unabhängig ausfüllen.

**Kann der Kunde seine Auswahl nach dem Senden ändern?**
Ja, er kann das Formular erneut ausfüllen. Sie sehen beide Einreichungen und können die letzte verwenden.

**Läuft der Link ab?**
Der Link funktioniert bis zum Ablaufdatum der Auswahl (falls festgelegt) oder bis zur Deaktivierung/Löschung der Auswahl.

**Verlangt der Link ein Passwort?**
Der Auswahllink verlangt kein separates Passwort. Wenn die Galerie ein Passwort hat, muss der Kunde es zuerst eingeben.

**Kann ich den Link deaktivieren?**
Ja, Sie können die Auswahlanfrage deaktivieren oder vollständig löschen. Der Link funktioniert dann nicht mehr.
