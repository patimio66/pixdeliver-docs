## Überblick

Gutscheincodes (Rabattcodes) ermöglichen es Ihnen, beim Bezahlen oder Verlängern einer Galerie in PixDeliver sowie bei der Aktivierung oder Verlängerung des Studio-Plans einen Rabatt zu erhalten. Gutscheine können einen prozentualen Rabatt, einen festen Rabattbetrag oder sogar eine kostenlose Galerie bieten.

**Wichtigste Informationen:**

- Gutscheine sind optional – Sie müssen sie nicht verwenden
- Ein Gutschein ist **mit einem bestimmten Plan verknüpft** (Plan pro Galerie oder Studio-Plan) und funktioniert nur bei diesem Plan
- Gutscheine gibt es in zwei Varianten:
  - **Einmalig** – können nur einmal im gesamten System verwendet werden
  - **Mehrfach verwendbar** – jeder Nutzer kann sie einmal verwenden (einmal pro Konto)
- Gutscheine funktionieren sowohl bei der ersten Zahlung als auch bei der Verlängerung/Erneuerung
- Die Groß- und Kleinschreibung im Code ist relevant

---

## Woher bekomme ich Gutscheincodes?

Gutscheincodes können aus verschiedenen Quellen stammen, zum Beispiel:

- Newsletter (abonnieren Sie den Newsletter!)
- Werbeaktionen (z. B. Black Friday)
- Marketingkampagnen

<Info>
  Folgen Sie PixDeliver und abonnieren Sie den Newsletter, um über Aktionen und Rabattcodes auf dem Laufenden zu bleiben!
</Info>

## Arten von Gutscheinen

### Prozentualer Rabatt

Reduziert den Preis um einen bestimmten Prozentsatz.

### Betragsrabatt

Zieht einen festen Betrag vom Preis ab.

### Kostenlose Galerie

Bietet 100 % Rabatt (die gewählte Galerieart im ersten Jahr kostenlos).

---

## Wie Sie einen Gutscheincode verwenden

### Beim Bezahlen einer Galerie (Plan pro Galerie)

**Anleitung:**

1. Gehen Sie zur Bezahlung der Galerie (Kleine oder Große Galerie)
2. Finden Sie das Feld **Gutscheincode** oder **Haben Sie einen Rabattcode?**
3. Klicken Sie, um das Feld aufzuklappen (falls eingeklappt)
4. Geben Sie den Code genau so ein, wie Sie ihn erhalten haben
   - Die Groß- und Kleinschreibung **ist relevant**: `PHOTO20` ≠ `photo20`
5. Klicken Sie auf **Anwenden** oder **Code verwenden**
6. Sie sehen den aktualisierten Preis mit Rabatt
7. Gehen Sie zur Zahlung

**Ergebnis:** Der Preis wird um den Rabattbetrag gesenkt. Sie zahlen weniger oder gar nichts (wenn der Gutschein 100 % Rabatt gewährt).

### Bei der Aktivierung oder Verlängerung des Studio-Plans

**Anleitung:**

1. Gehen Sie zu **Einstellungen** → **Studio-Plan**
2. Finden Sie auf dem Aktivierungs-/Verlängerungsbildschirm das Feld **Gutscheincode**
3. Geben Sie den Code ein und klicken Sie auf **Anwenden**
4. Der Preis des Studio-Plans wird um den Rabattbetrag gesenkt
5. Gehen Sie zur Zahlung

<Info>
  Gutscheine, die für den Studio-Plan ausgestellt wurden, funktionieren nicht beim Bezahlen einer einzelnen Galerie, und Gutscheine für einen Plan pro Galerie funktionieren nicht beim Studio-Plan. Das System weist Sie darauf hin, wenn der Code nicht zum gewählten Plan passt.
</Info>

<Tip>
  **Tipps:**

  - Kopieren Sie den Code und fügen Sie ihn ein, anstatt ihn abzutippen (so vermeiden Sie Fehler)
  - Achten Sie auf die Groß- und Kleinschreibung (Codes sind sogenannt case-sensitive)
</Tip>

## Einschränkungen von Gutscheinen

### Einmalig vs. mehrfach verwendbar

Das System unterscheidet zwei Arten von Gutscheinen:

- **Einmalig** – die erste Person, die den Code eingibt, verbraucht ihn für das gesamte System. Danach ist der Code inaktiv.
- **Mehrfach verwendbar** – jeder Nutzer kann den Code einmal verwenden. Wenn Sie einen solchen Code beim Studio-Plan oder bei einer Galerie verwenden, können Sie ihn kein zweites Mal auf Ihrem Konto verwenden – aber andere Nutzer können es weiterhin.

**Beispiel für einen mehrfach verwendbaren Gutschein:**

- Sie haben den Aktionscode `RABAT3` (mehrfach verwendbar)
- Sie verwenden ihn für die Galerie „Hochzeit Anna"
- Bei der Verlängerung dieser Galerie oder bei einer neuen Galerie auf demselben Konto **können Sie** denselben Code nicht erneut verwenden
- Der Code funktioniert weiterhin für andere Nutzer

### Verknüpfung mit einem Plan

Jeder Gutschein wird für einen bestimmten Plan ausgestellt:

- Plan Kleine Galerie
- Plan Große Galerie
- Studio-Plan

Der Code funktioniert ausschließlich beim Bezahlen des Plans, für den er ausgestellt wurde. Das System weist Sie vor der Zahlung auf eine Inkompatibilität hin.

### Gutscheine können nicht kombiniert werden

Sie können pro Transaktion nur einen Gutscheincode verwenden:

- ❌ Sie können nicht zwei Codes gleichzeitig anwenden
- ✅ Sie können den Code vor der Zahlung ändern (den vorherigen entfernen, einen neuen hinzufügen)

---

## Verschenken eines Gutscheincodes

Einige Codes können Sie an andere Nutzer verschenken (sofern sie mehrfach verwendbar sind).

**Wie Sie einen Code teilen:**

1. Kopieren Sie den Gutscheincode
2. Senden Sie ihn an einen Freund (E-Mail, Messenger, WhatsApp)
3. Geben Sie die Bedingungen an (Gültigkeitsdatum, Einschränkungen)

<Info>
  **Hinweis:** Einmalige Gutscheine (z. B. Geschenke) können Ihrem Konto zugeordnet sein und funktionieren nicht immer bei anderen Nutzern.
</Info>

## Fehlerbehebung

**Problem:** Der Gutscheincode funktioniert nicht. **Lösung:**

1. Prüfen Sie die Schreibweise – die Groß- und Kleinschreibung ist relevant
2. Stellen Sie sicher, dass keine Leerzeichen vor/nach dem Code stehen
3. Prüfen Sie, ob der Code nicht abgelaufen ist
4. Prüfen Sie, ob Sie ihn nicht bereits zuvor bei dieser Galerie verwendet haben
5. Stellen Sie sicher, dass der Code für den gewählten Plan gültig ist
6. Versuchen Sie, den Code erneut aus der Quelle zu kopieren
7. Aktualisieren Sie die Seite und versuchen Sie es erneut

**Problem:** Ich habe den Code eingegeben, aber der Rabatt ist nicht erschienen. **Lösung:**

1. Klicken Sie auf die Schaltfläche **Anwenden** (der Code muss bestätigt werden!)
2. Prüfen Sie die Meldung – wurde der Code akzeptiert?
4. Aktualisieren Sie die Seite und geben Sie den Code erneut ein

**Problem:** Das System sagt, dass der Code abgelaufen ist. **Lösung:**

- Der Code war nur für eine begrenzte Zeit gültig
- Prüfen Sie die Bedingungen des Codes (Gültigkeitsdatum)
- Suchen Sie nach neueren Codes (Newsletter, Social Media)
- Kontaktieren Sie die Person, die Ihnen den Code gegeben hat

**Problem:** Ich habe einen Code verwendet, aber den vollen Preis bezahlt. **Lösung:**

1. Prüfen Sie die Zahlungsbestätigungs-E-Mail
2. Sehen Sie nach, welcher Betrag abgebucht wurde
3. Wenn Sie trotz des angewendeten Codes den vollen Preis bezahlt haben:
   - Prüfen Sie, ob der Code vor der Zahlung sichtbar war
   - Kontaktieren Sie den Support mit einem Nachweis (Screenshot)
   - Geben Sie die Transaktionsnummer an

**Problem:** Ich möchte einen Code nach dem Bezahlen der Galerie verwenden. **Lösung:**

- Leider können Codes nach erfolgter Zahlung nicht mehr angewendet werden
- Codes funktionieren nur vor dem Bezahlen/Verlängern
- Denken Sie daran, den Code **vor** dem Übergang zur Zahlung einzugeben

## Häufig gestellte Fragen

**Kann ich einen Code bei einer bereits aktiven Galerie verwenden?** Nein. Codes funktionieren nur beim Bezahlen oder Verlängern einer Galerie (oder bei der Aktivierung/Verlängerung des Studio-Plans).

**Kann ich denselben Code bei mehreren Galerien verwenden?** Einen mehrfach verwendbaren Gutschein können Sie **einmal pro Konto** verwenden – unabhängig von der Anzahl Ihrer Galerien. Wenn Sie den Rabatt bei einer weiteren Galerie verwenden möchten, benötigen Sie einen anderen Code.

**Ist die Groß- und Kleinschreibung im Code relevant?** Ja, Codes sind sogenannt case-sensitive. `PHOTO20` ist ein anderer Code als `photo20`. Kopieren Sie den Code genau so, wie Sie ihn erhalten haben.

**Woher weiß ich, dass ein Code gültig ist?** Codes sollten ein Gültigkeitsdatum angegeben haben. Wenn Sie diese Information nicht haben, versuchen Sie es einfach – das System sagt Ihnen, ob der Code aktiv ist.

**Sind Rabattcodes immer verfügbar?** Nicht alle. Manche Codes sind zeitlich oder mengenmäßig begrenzt (z. B. die ersten 100 Verwendungen). Verwenden Sie Codes sofort!

**Bietet PixDeliver ein Empfehlungsprogramm an?** Derzeit haben wir kein offizielles Empfehlungsprogramm. Prüfen Sie die aktuellen Informationen im Verwaltungsbereich oder auf der Startseite. Funktionen können sich ändern.
